## Run Locally

Clone the project

```bash
  git clone https://github.com/piny4man/power-scouter.git
```

Go to the project directory

```bash
  cd power-scouter
```

(Optinal) Install `cargo make`

```bash
  cargo install cargo-make
```

Start the server

```bash
  cd front
  dioxus serve
```
or with `cargo make` from the root
```bash
  makers serve
```
